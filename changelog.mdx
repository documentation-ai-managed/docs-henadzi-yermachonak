<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Launched new Biothane leashes in multiple colors, featuring odor-free and waterproof material for ultimate durability.
- Added advanced product filters on the shop page, allowing you to sort by material, size, and style for faster browsing.

## Improvements

- Enhanced cart persistence so your selections remain saved across browser sessions.
- Updated smart ID tags with QR code integration linking directly to your pet's profile.

## Bug Fixes

- Fixed checkout flow issue where promo codes failed to apply on mobile devices.

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "bugfix"]}>

- Added subscription options for recurring orders of popular accessories.

## Improvements

- Optimized product images for faster page loads, reducing average load time by 40%.

## Bug Fixes

- Resolved search functionality bug that omitted results for "waterproof collar".
- Patched inventory sync issues preventing accurate stock display.

</Update>
